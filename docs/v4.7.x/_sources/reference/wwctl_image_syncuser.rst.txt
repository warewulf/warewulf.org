wwctl image syncuser
--------------------

Synchronize UIDs/GIDs from the server to an OS image

Synopsis
~~~~~~~~


Synchronize UIDs and GIDs from the server into an OS image, updating
/etc/passwd, /etc/group, and file ownerships within the image. Users and
groups that exist only in the image are preserved unless a UID/GID collision
is detected.

This command affects the image itself (a one-time operation at build/import
time). To also make host users available on provisioned nodes at runtime, add
the "syncuser" overlay to the node or profile.

::

  wwctl image syncuser [OPTIONS] IMAGE

Options
~~~~~~~

::

      --build   Build OS image after syncuser is completed
  -h, --help    help for syncuser
      --write   Synchronize UIDs/GIDs and write files in OS image (default true)
