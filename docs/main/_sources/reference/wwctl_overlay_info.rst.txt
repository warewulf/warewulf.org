wwctl overlay info
------------------

Show variables for a template file in an overlay

Synopsis
~~~~~~~~


This command will show the variables for a given template file in a given
overlay.

::

  wwctl overlay info [flags] OVERLAY_NAME FILE_PATH

Options
~~~~~~~

::

  -h, --help   help for info

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
