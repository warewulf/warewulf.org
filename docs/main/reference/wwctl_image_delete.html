wwctl image delete

Delete an imported image

Synopsis

This command will delete IMAGEs that have been imported into Warewulf.

wwctl image delete [OPTIONS] IMAGE [...]

Options

-h, --help   help for delete
-y, --yes    Set 'yes' to all questions asked
